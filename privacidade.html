Política de Privacidade

Última atualização: 2025

Esta Política descreve como o Big.Pet.Love Vídeos (“Aplicativo”) trata informações dos usuários.

1. Dados Coletados

O Aplicativo não coleta dados pessoais. Ele não solicita nome, e-mail, telefone, login, cookies, analytics ou qualquer outra informação identificável.

2. Uso do Aplicativo

Os vídeos publicados pelos usuários permanecem dentro do próprio sistema do TikTok, conforme regras e permissões da API utilizada.

3. Nenhum Dado Armazenado

O Big.Pet.Love não mantém banco de dados, não armazena informações, não cria perfis e não rastreia atividades.

4. Segurança

Como nenhum dado é coletado ou armazenado, não há risco de exposição de informações pessoais.

5. Transferência de Dados

Não há compartilhamento ou transferência de dados para terceiros.

6. Base Legal

Esta Política segue a legislação brasileira aplicável, com jurisdição em São Paulo – Brasil.
